# Image Studio Feature

This directory contains the implementation of the Image Studio feature, a playground for text-to-image and image-to-image generation.

## Structure

- `page.tsx`: The main page component implementing the UI using shadcn/ui components.
- `readme.md`: (In `features/image-studio/`) Feature documentation and requirements.
- `prototype.html`: (In `features/image-studio/`) HTML prototype.

## Tech Stack

- **Framework**: Next.js (App Router)
- **UI**: shadcn/ui + Tailwind CSS
- **Icons**: Lucide React
- **State**: React local state (will migrate to Zustand for global state)

## Usage

Navigate to `/features/image-studio` to access the playground.
